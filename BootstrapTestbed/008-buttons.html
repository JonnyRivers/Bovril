Buttons Styles


To achieve the button styles above, Bootstrap has the following classes:

  • .btn-default
  • .btn-primary
  • .btn-success
  • .btn-info
  • .btn-warning
  • .btn-danger
  • .btn-link

Buttons Sizes

The classes that define sizes are:

  • .btn-lg
  • .btn-md
  • .btn-sm
  • .btn-xs
